## üîê Interceptores Autom√°ticos

### **Interceptor de Request**
```javascript

  const user = JSON.parse(localStorage.getItem('empathica_user') || '{}');
  if (user && user.token) {
    config.headers.Authorization = `Bearer ${user.token}`;
  }
  return config;
});
```

**Funci√≥n**: Agrega autom√°ticamente el token de autenticaci√≥n a todas las peticiones.

### **Interceptor de Response**
```javascript

  (response) => response,
  (error) => {
    if (error.response && error.response.status === 401) {
      // Token expirado o inv√°lido
      localStorage.removeItem('empathica_user');
```

**Funci√≥n**: Maneja autom√°ticamente errores de autenticaci√≥n (401) y redirige al login.
